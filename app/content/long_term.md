The longer-term vision is a data and AI capability that compounds — where every new project makes the next one cheaper, because the inputs, tooling, and conventions are already in place. The goal across 6 to 24 months is not to build something impressive in isolation, but to make sure that when an analyst has an idea on a Tuesday, the path from idea to a credible answer is measured in hours rather than weeks.

## Capability buckets

- **Alternative data integration.** Ingest non-traditional sources — satellite imagery, web data, transcripts, vendor feeds — into the same feature store the rest of the firm already uses, and evaluate each one honestly against existing strategies before declaring it useful.
- **AI-assisted research.** LLM-based agents that read filings, transcripts, and broker reports and surface what is new, contradictory, or numeric. Analysts stay in the driver's seat; the model handles the reading volume that no human team can keep up with.
- **Backtest and signal infrastructure.** A lightweight, reproducible framework so any researcher can prototype an idea, run it against history, and share the result without rebuilding plumbing each time. Reproducibility is the feature.
- **Risk and attribution analytics.** Live dashboards that show what the portfolio is actually exposed to — by factor, by theme, by correlated cluster — beyond the line items in a position file.
- **Knowledge capture.** A queryable record of every research decision, post-mortem, and trade rationale, so institutional memory stops walking out the door when people leave.

The long-term goal is to make Merewether's investment process *legibly* better — not flashier, just clearer and faster.
